canvas 绘制五角星

Created at 2017-10-28 Updated at 2017-10-28

思路

五角星的顶点落在两个圆（同心）上，外(凸)顶点落在外圆，内(凹)顶点在内圆上。

记住几个关键点

  • 外圆半径–R
  • 内圆半径–r
  • 外顶点坐标(x,y)
x:Math.cos((18+72*i)/180*Math.PI)*R;
y:Math.sin((18+72*i)/180*Math.PI)*R

  • 内顶点坐标(x,y)

    x:Math.cos((54+72i)/180Math.PI)r;
    y:Math.sin((54+72    i)/180Math.PI)r

以上五角星内外顶点公式得来，过程这里不做详细介绍。

方法

下面是封装好的使用canvas绘制五角星的方法：

function drawStar(R,r,x,y,ctx,fillStyle) {
    //  r、R分别是内外圆半径
    // x、y分别是水平、垂直偏移量

    ctx.beginPath();
    for(var i = 0; i < 5; i ++) {
        // 外顶点(x1,y1)  内顶点(x2,y2)
        var x1 = Math.cos((18+72*i)/180*Math.PI)*R,
            y1 = Math.sin((18+72*i)/180*Math.PI)*R,
            x2 = Math.cos((54+72*i)/180*Math.PI)*r,
            y2 = Math.sin((54+72*i)/180*Math.PI)*r;

        ctx.lineTo(x1 + x, y1 + y);
        ctx.lineTo(x2 + x, y2 + y);
    }

    ctx.closePath();
    ctx.fillStyle = fillStyle;
    ctx.fill();
}

例子 – 绘制指定数量(n)随机分布的星星

function drawRandomStars(ctx,n) {
    // 绘制矩形背景
    ctx.fillStyle = '#382a6c';
    ctx.fillRect(0,0,180,220);

    // 绘制随机分布的星星
    // 调用上面的函数drawStar
    for(var i = 0; i < n; i ++) {
        ctx.save(); // 保存canvas状态，进行下一步的绘制（连续绘制）
        var R = Math.random() * 4 + 1, // 外圆半径，内圆半径取R/2
            x = Math.random() * 160 + 20, // 水平偏移量
            y = Math.random() * 200 + 20, // 垂直偏移量
            fillStyle = '#fff'; // 给每个星星填充白色

        this.drawStar(R, R/2, x, y, ctx, fillStyle);
    }
}
